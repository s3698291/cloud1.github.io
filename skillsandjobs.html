Skills and Jobs

The first candidate should be an UX/UI designer since there are no designers listed in roles section. They will be responsible for the usability aspect of the software to ensure that it is easy for users to use and navigate through the software. Creativity is a required skill since the designer has to think of a simple but effective design that not only can capture the client’s attention but it should be straightforward as well. If the users can figure out how to use the software in a short amount of time, it means that their task is achieved. Communication skills are important because designers will required to interact with clients and software developers and engineers to ensure that both parties can achieve their needs. Teamwork is a necessity as well since each staff member will be working in groups and the skill ensures that each member can collaborate without too many issues or miscommunication. Other skills include Java and graphic design because the software will be written in Java, although designers will only work on front end part. Graphic design skills would be very helpful since designers can design icons to be displayed in the software.

The second candidate would be a database administrator because containing all of the data in a database is essential for any company. The database administrator can organize files as well and ensure that data inserted into database is correct. Communication skills is key because the administrator must ensure that they can get updated about new information that they have to insert into the database and must always prevent miscommunication among other employees. Organisational skills is another important skill since their job is to organize all of the information they receive regarding the clients and software itself. SQL skills are a necessity since they will be using that language to create the required database. Database administrators will be judged based on how familiar they are with using SQL and how they implement it into creating databases. They must be familiar with using Microsoft Windows as well because the software will be operating using Windows. Being knowledgeable at both set of skills would suffice in terms of IT skills. The last required skill would be problem solving because solving new problems is inevitable. It will be beneficial and can contribute a lot of help, such as being able to find new efficient ways.

The third candidate can join the team is a marketing specialist. As the role of a marketing specialist might sounds, it is not an easy job. A marketing generalist knows the best way to advertise Cloud 1 service. Advertisement is an important component in determining a success of a new start up business. This is because advertisement acts as a tool for the owner of the brand able to introduce the product or service to the public.Knowing a right target audience and the effective ways to advertise a product can be a difficult challenge. Hence, a marketing generalist is required as they already have all of the knowledge such as: how to market a product correct, what is the target audiences, etc. Not only they have the knowledges, but they also have experiences and relationship with advertisement companies such as TV or companies that specialize in printed advertisements. Additionally, a marketing generalist can help the company to save a fortune in advertisement, as the generalist knows exactly how and when to target the right audiences, rather than spending a large amount of money advertising to the incorrect targets and places. The aim of Cloud 1 service is to create a better and a much more security cloud systems to the public and being internationally recognised. By investing the money into advertisements of Cloud 1 services, this might improve chances of Cloud 1 being recognised worldwide. In order to achieve this, a marketing generalist may have the answer of how this aim can be achieved.

The fourth candidate is a sale representative. A representative would be suitable as they can bring new customers to Cloud 1 by introducing Cloud 1 service. Advertisement is essential to Cloud 1, especially, during the first initial launch of the business. A sale representative will have good communication skills, this will allow them to communicate to the company and attract the customers to use Cloud 1 service at the same time. Furthermore, a sale representative must have previous experience in this filed and a friendly personality. Friendly personality is will be much easier to attract customers and increase the chances of the customers to use the product, hence, increasing sales for the company. Previous experiences are also required such as ability to scale business, be familiar with buyers and must be a team player. Ability to work well in a team is an essential skill to have, as the whole business depend on the sale representatives’ abilities to grab customers’ attentions and increasing revenue for the whole company. If the sale representatives are not a team player, this may cause misunderstanding and miscommunications between the developers and the sale representatives. Additionally, this sale representative must be culturally fit. Cultural fit means that the sale representative is not bias, ability to fit into a team with different cultural backgrounds and able to interact to different customers with different cultural backgrounds. Therefore, more the company would able to target a lot more customers with different backgrounds, which increase the chance of Cloud 1 being internationally recognised.
